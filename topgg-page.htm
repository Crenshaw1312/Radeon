Radeon Development

Radeon is a multi-functional moderation bot with an anti-raid system! While still a small bot, it has a powerful and up-to-date moderation system with active development on the anti-raid system. Any and all support is appreciated!

Commands

All commands can be triggered by using the prefix or by mentioning Radeon (e.g. @Radeon help). Some commands can also be used in DMs (shown in the help message). DM commands do not require a prefix, just the command and arguments (e.g. help ping).
Commands with required arguments have <ab>, optional arguments have [ab], and arguments where you pick one use pipes a|b.

General

  • help [Category|Command] - Sends the help embed or help on a specified command.
  • invite - Sends Radeon's invite links.
  • privacy - Sends Radeon's privacy policy (information on data collection and usage).
  • serverinfo - Sends information about the server.
  • userinfo [User:Mention/ID] - Sends information about a specified Member, or the triggering Member if none is specified.
  • roleinfo <Role:Name/Mention/ID> - Sends information about a specified role.
  • role <SubCommand> <...args> - Role Tools: a set of powerful role commands (see help role for more).
  • embed [...Flags] - Creates an embed using flags (see help embed for flags).
  • jsonembed <JSON:Text> - Creates an embed using JSON text.
  • editembed <JSON:Text> - Edits an existing embed message sent by Radeon using JSON text.

    • Maintenance

  • ping - Sends Radeon's ping!
  • status - Sends Radeon's overral status
  • prefix - Sends the prefix(es) for Radeon
  • shard [Guild:ID] - Sends the shard information for that server or a specified server.
  • permissions <...args> - Permissions Tools: a set of readonly permission commands (see help permissions for more)

    • Moderation

  • slowmode <Time:Duration|off> [Reason:Text] - Sets the channel slowmode to a specified duration or off.
  • clean <Amount:Number> [...Flags] - Deletes a number of messages with optional flags (see help clean for more).
  • mute <User:Mention/ID> [Time:Duration] <Reason:Text> - Mutes a specified Member (optional duration).
  • unmute <User:Mention/ID> [Reason:Text] - Unumutes a muted Member (optional reason).
  • kick <User:Mention/ID> [Reason:Text] - Kicks a Member from the server.
  • ban <User:Mention/ID> <Reason:Text> [-dd <Number>] - Bans a Member from the server. Optional deletion days flag to delete Member messages (max. 7).
  • unban <User:ID> [Reason:Text] - Unbans a banned User (optional reason).

    • Server Settings

  • setprefix <Prefix:Text|reset> - Changes Radeon's prefix for the server.
  • muterole [Role:Name/Mention/ID|reset] - Sends the current mute role for the server and allows for a new one to be set.
  • modlogs <SubCommand> <arg> - Modlogs config commands for managing logs (see help modlogs for more).
  • automod <SubCommand> <...args> - Automod config commands for managing automod triggers and events (see help automod for more).
  • override <SubCommand> <...args> - Channel and command override manager (see help override for more).
